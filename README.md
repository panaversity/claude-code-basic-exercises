## ğŸ§  The Problem-Solving Framework

Use this for EVERY exercise:

1. **Define** â€” What exactly am I trying to accomplish?
2. **Gather** â€” What files/data does Claude need?
3. **Specify** â€” Describe the desired outcome, constraints, and format
4. **Execute** â€” Run it with Claude Code or Cowork
5. **Verify** â€” Does the output match what I asked for?
6. **Iterate** â€” What would I change? Run it again.
7. **Reflect** â€” What did I learn about specifying problems clearly?
